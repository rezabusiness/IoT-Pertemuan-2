# IoT-Pertemuan-2

Tugas Pekan 2 – Mata Kuliah Teknologi IoT

Proyek ini dibuat menggunakan ESP32 dengan tujuan untuk:

Membaca data suhu dan kelembapan dari sensor DHT22
Mengirimkan data ke ThingSpeak secara berkala
Menghubungkan perangkat ke jaringan WiFi

Fitur yang tersedia:
1. Monitoring suhu dan kelembapan secara real-time
2. Pengiriman data ke ThingSpeak setiap 15 detik
3. LED akan menyala ketika suhu melebihi 30°C

Perangkat yang digunakan:
1. ESP32
2. Sensor DHT22
3. LED
4. Resistor 220 ohm dan 1k ohm
5. Kabel jumper

Proyek ini disimulasikan menggunakan platform Wokwi.
